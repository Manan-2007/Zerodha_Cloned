## 📖 About

A static clone of the **Zerodha** homepage — India's largest stock brokerage platform. This is a UI replication exercise focused on faithfully recreating Zerodha's clean, minimal landing page: the logo, navigation links, hero investment graphic, headline text, CTA button, and footer — using only HTML and CSS.

> ⚠️ **Disclaimer:** This is a non-commercial learning project. Zerodha, its branding, logo, and assets are property of **Zerodha Broking Ltd.** All rights reserved.

---

## ✨ What's Replicated

| Section | Details |
|---|---|
| **Navigation** | Zerodha logo (left), nav links: Signup, About, Products, Pricing, Support, icon (right) |
| **Hero** | Large platform screenshot/graphic image at 55.5% width |
| **Headline** | "Invest in everything" bold text |
| **Subtext** | Platform description paragraph |
| **CTA Button** | "Sign up for free" primary action button |
| **Footer** | Copyright notice |
